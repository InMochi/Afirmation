Media Queries dalam CSS

Media Queries adalah fitur dalam CSS yang memungkinkan kita untuk menerapkan gaya yang berbeda berdasarkan ukuran layar atau perangkat yang digunakan.

1. Penggunaan Dasar Media Queries

Media Queries digunakan dengan aturan @media untuk menentukan gaya CSS yang hanya berlaku pada kondisi tertentu, misalnya ketika layar memiliki lebar maksimum atau minimum tertentu.

Contoh perubahan warna berdasarkan ukuran layar:

Ubah ukuran layar!
    
.responsive-box {
    width: 100%;
    height: 100px;
    background-color: lightblue;
    text-align: center;
    line-height: 100px;
    font-weight: bold;
}

@media (max-width: 768px) {
    .responsive-box {
        background-color: orange;
        color: white;
    }
}

@media (max-width: 480px) {
    .responsive-box {
        background-color: red;
        color: white;
    }
}

2. Mengubah Layout dengan Media Queries

Kita juga bisa mengatur ulang tata letak elemen berdasarkan ukuran layar. Contoh di bawah ini menunjukkan bagaimana kolom yang awalnya sejajar berubah menjadi vertikal di layar kecil.

Kolom 1
Kolom 2
    
.responsive-layout {
    display: flex;
    gap: 10px;
}

.box {
    flex: 1;
    height: 100px;
    background-color: green;
    color: white;
    text-align: center;
    line-height: 100px;
    font-weight: bold;
}

@media (max-width: 768px) {
    .responsive-layout {
        flex-direction: column;
    }
}
