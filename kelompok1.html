Animasi dan Transisi dalam CSS

Animasi dan transisi adalah dua teknik dalam CSS yang digunakan untuk memberikan efek perubahan tampilan pada elemen tanpa memerlukan JavaScript.

1. Transisi (CSS Transition)

Transisi digunakan untuk memberikan efek perubahan properti CSS secara bertahap. Ini memungkinkan perubahan yang lebih halus dibandingkan dengan perubahan langsung.

Contoh penggunaan transisi:

Hover saya!
    
.transition-box {
    width: 150px;
    height: 50px;
    background-color: lightblue;
    text-align: center;
    line-height: 50px;
    border-radius: 5px;
    transition: background-color 0.5s, transform 0.3s;
}

.transition-box:hover {
    background-color: blue;
    color: white;
    transform: scale(1.1);
}

2. Animasi (CSS Animation)

Animasi memungkinkan elemen mengalami perubahan dalam rentang waktu tertentu menggunakan keyframes.

Contoh animasi dengan keyframes:

Saya bergerak!
    
@keyframes moveBox {
    0% { transform: translateX(0); }
    50% { transform: translateX(100px); }
    100% { transform: translateX(0); }
}

.animation-box {
    width: 150px;
    height: 50px;
    background-color: orange;
    text-align: center;
    line-height: 50px;
    border-radius: 5px;
    animation: moveBox 2s infinite;
}
