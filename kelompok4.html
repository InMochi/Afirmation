Responsive Typography

Responsive Typography adalah teknik dalam CSS untuk membuat ukuran teks yang menyesuaikan dengan ukuran layar. Ini penting untuk memastikan teks tetap nyaman dibaca di berbagai perangkat.

1. Menggunakan Unit Fleksibel

Unit yang sering digunakan dalam responsive typography:

  • em: Berdasarkan ukuran font elemen induk.
  • rem: Berdasarkan ukuran font root (`html`).
  • vw (viewport width): Berdasarkan lebar layar.
  • %: Berdasarkan elemen induk.

Contoh CSS:

    
/* Ukuran font berbasis viewport */
.responsive-text {
    font-size: 5vw;
}

/* Menggunakan rem untuk ukuran yang lebih stabil */
.rem-text {
    font-size: 2rem;
}

/* Menggunakan media queries */
@media (max-width: 768px) {
    .responsive-text {
        font-size: 3vw;
    }
    .rem-text {
        font-size: 1.5rem;
    }
}

Hasilnya dalam HTML:

Teks ini menggunakan font-size berbasis viewport (5vw).

Teks ini menggunakan font-size berbasis rem (2rem).

2. Menggunakan CSS Clamp()

clamp() memungkinkan kita menetapkan ukuran teks minimum, ideal, dan maksimum.

Contoh CSS:

    
.clamp-text {
    font-size: clamp(1rem, 2.5vw, 3rem);
}

Hasilnya dalam HTML:

Teks ini menggunakan clamp() untuk menyesuaikan ukuran.
