CSS Positioning dan Z-Index

**CSS Positioning** digunakan untuk mengatur posisi elemen dalam halaman web. CSS menyediakan beberapa jenis posisi yang berbeda untuk mengontrol tata letak elemen.

1. Jenis Positioning dalam CSS

  • Static: Default posisi elemen (mengikuti alur dokumen).
  • Relative: Bergerak relatif terhadap posisi aslinya.
  • Absolute: Posisi berdasarkan elemen induk yang memiliki posisi selain `static`.
  • Fixed: Tetap di tempat meskipun halaman di-scroll.
  • Sticky: Mengikuti scroll tetapi tetap di posisi tertentu saat mencapai batas.

Contoh CSS Positioning:

    
/* Relative Position */
.relative-box {
    position: relative;
    left: 20px;
    top: 10px;
    background-color: lightblue;
    padding: 10px;
}

/* Absolute Position */
.absolute-box {
    position: absolute;
    top: 50px;
    right: 20px;
    background-color: lightcoral;
    padding: 10px;
}

/* Fixed Position */
.fixed-box {
    position: fixed;
    bottom: 10px;
    right: 10px;
    background-color: lightgreen;
    padding: 10px;
}

/* Sticky Position */
.sticky-box {
    position: sticky;
    top: 0;
    background-color: yellow;
    padding: 10px;
}

2. Z-Index

Z-Index menentukan urutan tumpukan elemen. Semakin tinggi nilainya, semakin di depan elemen tersebut.

Contoh CSS Z-Index:

    
.box1 {
    position: absolute;
    top: 100px;
    left: 50px;
    width: 100px;
    height: 100px;
    background-color: red;
    z-index: 1;
}

.box2 {
    position: absolute;
    top: 120px;
    left: 70px;
    width: 100px;
    height: 100px;
    background-color: blue;
    z-index: 2;
}

Hasilnya dalam HTML:

Relative Box
Absolute Box
Fixed Box
Sticky Box (Scroll untuk melihat efeknya)
