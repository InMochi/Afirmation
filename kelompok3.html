Sass dalam CSS

Sass (Syntactically Awesome Stylesheets) adalah preprocessor CSS yang menambahkan fitur seperti variabel, nesting, mixin, dan lain-lain. Dengan menggunakan Sass, kita bisa menulis CSS yang lebih efisien dan mudah dikelola.

1. Instalasi Sass

Untuk menggunakan Sass, kita perlu menginstalnya terlebih dahulu. Jika menggunakan Node.js, kita bisa menginstalnya dengan perintah berikut: 

npm install -g sass

Setelah itu, kita bisa mengonversi file .scss menjadi .css dengan perintah: 

sass style.scss style.css

2. Contoh Penggunaan Sass

Berikut adalah contoh penggunaan Sass:

Kode SCSS:

    
// Variabel
$primary-color: #3498db;
$secondary-color: #2ecc71;

// Mixin untuk tombol
@mixin button-style {
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
}

// Style elemen
.sass-box {
    width: 200px;
    height: 100px;
    background-color: $primary-color;
    text-align: center;
    line-height: 100px;
    color: white;
    font-weight: bold;
    
    &:hover {
        background-color: $secondary-color;
    }
}

.sass-button {
    @include button-style;
    background-color: $primary-color;
    color: white;
    
    &:hover {
        background-color: $secondary-color;
    }
}

Hasilnya dalam HTML:

Hover saya!

3. Keuntungan Menggunakan Sass

  • Variabel: Memudahkan pengelolaan warna dan ukuran.
  • Nesting: Mempermudah penulisan CSS dengan struktur yang lebih jelas.
  • Mixin: Membantu membuat reusable styles.
  • Modularisasi: Bisa membagi file menjadi bagian-bagian kecil dengan @import.

Sass Nesting

Nesting dalam Sass memungkinkan kita menulis CSS dengan struktur yang lebih rapi dan mengikuti hierarki HTML. Ini membuat kode lebih mudah dibaca dan dikelola dibandingkan dengan CSS biasa.

1. Contoh Penggunaan Nesting

Berikut adalah contoh kode SCSS yang menggunakan nesting:

Kode SCSS:

    
/* Variabel */
$primary-color: #3498db;
$secondary-color: #2ecc71;

/* Style dengan Nesting */
.card {
    width: 250px;
    border-radius: 10px;
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
    overflow: hidden;
    
    .card-header {
        background-color: $primary-color;
        color: white;
        padding: 15px;
        text-align: center;
    }
    
    .card-body {
        padding: 15px;
        font-size: 14px;
    }

    .card-footer {
        background-color: $secondary-color;
        padding: 10px;
        text-align: center;
        color: white;
    }
}

Judul Card
Ini adalah konten dalam card.
